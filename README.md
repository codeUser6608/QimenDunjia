# 奇门遁甲 H5小程序

> 中国古代术数模拟应用 - 探索奇门遁甲的奥秘

## 项目简介

奇门遁甲H5小程序是一个基于Vue 3 + Vite开发的移动端优先的术数模拟应用，旨在将深奥的奇门遁甲术数体系通过游戏化、可视化的方式呈现，降低学习门槛，让传统文化得以传承。

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **构建工具**: Vite 5
- **状态管理**: Pinia
- **路由管理**: Vue Router 4
- **样式预处理**: SCSS
- **日期处理**: Day.js

## 核心功能

### 1. 知识学习
- 基础知识：天干地支、五行、八卦
- 八门详解：开、休、生、伤、杜、景、死、惊
- 九星详解：天蓬、天任、天冲等九星
- 八神详解：值符、腾蛇、太阴等八神

### 2. 排盘推演
- 时间排盘：根据年月日时自动排盘
- 空间排盘：根据方位进行排盘
- 手动排盘：用户手动调整四盘布局
- 四盘系统：天盘、地盘、人盘、神盘

### 3. 事件推演
- 事业推演：分析事业发展吉凶
- 婚姻推演：分析婚姻运势
- 出行推演：分析出行吉凶
- 投资推演：分析投资时机

### 4. 每日运势
- 奇门日历：每日奇门格局
- 时辰吉凶：每个时辰的吉凶分析
- 宜忌事项：每日宜做什么、忌做什么

### 5. 挑战模式
- 排盘挑战：根据条件完成正确排盘
- 解盘挑战：分析给定格局的含义
- 案例分析：分析历史奇门案例

## 功能说明

### 四盘系统

奇门遁甲的核心是四盘系统：

1. **天盘**：九星排布
   - 天蓬、天任、天冲、天辅、天英、天芮、天柱、天心、天禽

2. **地盘**：九宫天干
   - 坎一、坤二、震三、巽四、中五、乾六、兑七、艮八、离九

3. **人盘**：八门排布
   - 开门、休门、生门、伤门、杜门、景门、死门、惊门

4. **神盘**：八神排布
   - 值符、腾蛇、太阴、六合、白虎、玄武、九地、九天

### 学习体系

- **入门**：天干地支、五行、八卦（1-2周）
- **进阶**：八门、九星、八神（2-3周）
- **高级**：排盘方法、格局分析（3-4周）
- **精通**：实战应用、案例解析（4-6周）
